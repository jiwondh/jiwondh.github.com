# [C++] 입력 개수 모를 때 입력 받기

입력 개수가 주어지지 않고 **입력이 끝날 때까지 입력 받아야 될 때** 사용하시면 될 것 같아요~

### 예제

{% highlight c++ %}

# include <iostream>

using namespace std;

int main() {
​	int n;

​	do {
​		cin >> n;
​		cout << n << endl;
​	} while (getc(stdin) == ' ');

​	cout << "end";
​	return 0;
}

{% endhighlight %}
