## 문제

- 양팔 저울과 몇 개의 추가 주어졌을 때, 이를 이용하여 **입력으로 주어진 구슬의 무게를 확인**할 수 있는지를 **결정**

### 예제

```
추 1g과 4g으로 만들 수 있는 무게
- DP(0, 1) => 1
- DP(1, 4) => 4, 3(4-1), 5(4+1)

추 1g, 4g, 7g으로 만들 수 있는 무게

- DP(0, 1) => 1
- DP(1, 4) => 4, 3(4-1), 5(4+1)
- DP(4, 7) => 7, 3(7-4), 11(7+4)
```
